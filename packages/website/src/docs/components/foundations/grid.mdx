# Grid

`Grid` is a primitive layout component that can arrange content in a grid configuration.

## Usage

```jsx live
import { Grid } from "@wp-g2/components"

function Example() {
  return (
    <Grid columns={3}>
      <View css={[ui.background.blue]}>
        <Text>Ana</Text>
      </View>
      <View css={[ui.background.blue]}>
        <Text>Elsa</Text>
      </View>
      <View css={[ui.background.blue]}>
        <Text>Olaf</Text>
      </View>
    </Grid>
  )
}
```

## Props

| Name            | Type                      | Default | Description                                           |
| --------------- | ------------------------- | ------- | ----------------------------------------------------- |
| align           | `string`                  |         | Adjusts the block alignment of children.              |
| columns         | `Array<number>`, `number` | `2`     | Adjusts the number of columns of the `Grid`.          |
| isInline        | `boolean`                 | `false` | Changes the CSS display from `grid` to `inline-grid`. |
| justify         | `string`                  |         | Adjusts the inline alignment of children.             |
| rows            | `Array<number>`, `number` |         | Adjusts the number of rows of the `Grid`.             |
| templateColumns | `string`                  |         | Adjusts the CSS grid `template-columns`.              |
| templateRows    | `string`                  |         | Adjusts the CSS grid `template-rows`.                 |
