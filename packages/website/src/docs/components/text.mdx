# Text

`Text` renders text using the library's typography system.

## Usage

```jsx live
import { Text } from "@wp-g2/components"

function Example() {
  return <Text>Where the north wind meets the sea...</Text>
}
```

## Props

| Name          | Type               | Default | Description                                                          |
| ------------- | ------------------ | ------- | -------------------------------------------------------------------- |
| align         | `string`           |         | Adjusts the text alignment.                                          |
| as            | `string`           | `span`  | Renders as another HTML element (or React component).                |
| color         | `string`           |         | Adjust the text color.                                               |
| display       | `string`           |         | Adjusts the text CSS `display`.                                      |
| ellipsis      | `string`           | `...`   | The ellipsis string when `truncate` is set.                          |
| ellipsizeMode | `string`           | `auto`  | Determines where to truncate. `truncate` must first be set.          |
| limit         | `number`           | `0`     | Determines the max characters when `truncate` is set.                |
| isBlock       | `boolean`          |         | Sets `Text` to have `display: block`.                                |
| lineHeight    | `number`           |         | Adjusts all text line-height based on the typography system.         |
| size          | `number`, `string` |         | Adjusts all text size based on the typography system.                |
| truncate      | `boolean`          |         | Enables text truncation.                                             |
| numberOfLines | `boolean`          |         | Determines how many lines to truncate. `truncate` must first be set. |
| upperCase     | `boolean`          |         | Uppercases the text content.                                         |
| variant       | `string`           |         | Adjusts style variation of the text.                                 |
| weight        | `number`,`string`  |         | Adjusts font-weight of the text.                                     |

### align

| Value  | Description                |
| ------ | -------------------------- |
| left   | Aligns text to the left.   |
| center | Aligns text to the center. |
| right  | Aligns text to the right.  |

```jsx live
import { Text } from "@wp-g2/components"

function Example() {
  return <Text align="left">Where the north wind meets the sea...</Text>
}
```

### size

`Text` can render a wide range of font sizes, which are automatically calculated and adapted to the typography system. The `size` value can be a system preset, a `number`, or a custom unit value (`string`) such as `30em`.

#### Preset

| Value       | Description                               |
| ----------- | ----------------------------------------- |
| largeTitle  | Renders an effective font-size of `28px`. |
| title       | Renders an effective font-size of `20px`. |
| body        | Renders an effective font-size of `13px`. |
| subheadline | Renders an effective font-size of `12px`. |
| footnote    | Renders an effective font-size of `11px`. |
| caption     | Renders an effective font-size of `10px`. |

```jsx live
import { Text } from "@wp-g2/components"

function Example() {
  return <Text size="largeTitle">Where the north wind meets the sea...</Text>
}
```

#### Custom

```jsx live
import { Text } from "@wp-g2/components"

function Example() {
  return <Text size={20}>Where the north wind meets the sea...</Text>
}
```

### truncate

When `truncate` is set,we are able to truncate the long text in a variety of ways.

```jsx live
import { Text } from "@wp-g2/components"

function Example() {
  return (
    <Text truncate>
      Where the north wind meets the sea, there's a river full of memory. Sleep,
      my darling, safe and sound, for in this river all is found. In her waters,
      deep and true, lay the answers and a path for you. Dive down deep into her
      sound, but not too far or you'll be drowned
    </Text>
  )
}
```

#### ellipsizeMode

For example, we can truncate text right in the middle. To do this, we need to set `ellipsizeMode` to `middle` and a text `limit`. For more information about truncation, check out [`Truncate`](/components/truncate).

```jsx live
import { Text } from "@wp-g2/components"

function Example() {
  return (
    <Text truncate ellipsizeMode="middle" limit={40}>
      Where the north wind meets the sea, there's a river full of memory. Sleep,
      my darling, safe and sound, for in this river all is found. In her waters,
      deep and true, lay the answers and a path for you. Dive down deep into her
      sound, but not too far or you'll be drowned
    </Text>
  )
}
```

### variant

| Value | Description         |
| ----- | ------------------- |
| muted | Renders muted text. |

```jsx live
import { Text } from "@wp-g2/components"

function Example() {
  return <Text variant="muted">Where the north wind meets the sea...</Text>
}
```
